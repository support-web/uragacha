# 占いガチャ

占いポータルサイト向けの独立したガチャコンテンツページです。

## 機能

- ガチャマシン風のインタラクティブUI
- 6段階の運勢（大吉・中吉・吉・末吉・凶・大凶）
- 重み付き抽選による運勢決定
- 6種類の占い種別（総合運・恋愛運・金運・仕事運・健康運・対人運）
- ラッキーカラー・ラッキーアイテム・ラッキーナンバーの表示
- レスポンシブデザイン対応
- アニメーション演出

## ファイル構成

```
uragacha/
├── index.html      # メインページ
├── css/
│   └── style.css   # スタイルシート
├── js/
│   └── gacha.js    # ガチャロジック
└── README.md
```

## 使い方

1. `index.html` をブラウザで開く
2. ガチャマシンの赤いボタンをクリック
3. 今日の運勢を確認！

## 技術仕様

- HTML5 / CSS3 / JavaScript (ES6+)
- 外部ライブラリ不要（バニラJS）
- Google Fonts（Zen Maru Gothic, Noto Sans JP）

## カスタマイズ

### 運勢の確率調整

`js/gacha.js` の `fortuneData` オブジェクト内の `probability` 値を変更することで、各運勢の出現確率を調整できます。

### メッセージの追加

各運勢の `messages` 配列に新しいメッセージを追加できます。

### ラッキーアイテム/カラーの追加

`luckyItems` / `luckyColors` 配列に新しい項目を追加できます。
